## Why the Change?

The old AGENTS.md was ~2,600 words (~3,900 tokens) and required agents to load everything upfront, even when they only needed "how do I run this command?" or "what's the DB schema?"

**New structure:**
- **Token savings:** 83% reduction (10,800 → 1,800 tokens for typical tasks)
- **Lazy loading:** Load only what you need via quick-refs
- **Clear navigation:** Decision tree tells you which doc to load next
